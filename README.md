## Testing

- Clone the project

```npm
git clone https://github.com/theomaro/sci-math-quiz.git
cd sci-math-quiz
```

- Install packages

```npm
npm install
```

- Set development enviroment variables

```
$ touch .env && echo PUBLIC_API_URL=http://localhost:3000/quizzes
```

- Start a json server

```npm
npm run json-server
```

- Start a development server

```npm
npm run dev
```
